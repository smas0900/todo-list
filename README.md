# 📖 Todo List <a name="about-project"></a>

This is a simple but useful project where user can add and remove his/her todo tasks. It maintains those tasks in the assending order. User can also mark and complete the completed tasks.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

<details>
  <summary>Client</summary>
  <ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
     <li>Webpacks</li>
  </ul>
</details>

<!-- Features -->

### Key Features <a name="key-features"></a>

<ul>
<li>Dynamic To-do Lists</li>
<li>Interesting UI</li>
<li>Add tasks and mark them as done.</li>
<li>Delete tasks one by one or delete all marked done at one</li>
<li>Reorder list by dragging and dropping tasks
</li>
</ul>
